# 1— 知识扩展
数组动态初始化 可以用 int[] arr = [5] 这种方式简写吗
### 0.1— 为什么 `int[] arr = [5]` 不行？

- Java 的语法规则中，数组初始化必须显式使用 `new 类型[]`（动态 / 静态）或 `{}`（简化静态初始化），单独的 `[5]` 不是合法的数组字面量；
- 这种 `[5]` 的写法更接近 JavaScript 等脚本语言的数组创建方式，但 Java 作为强类型语言，有严格的语法规范，不支持这种简写。

### 0.2— 总结

1. Java 中**不支持** `int[] arr = [5]` 这种简写，这是语法错误；
2. 创建包含元素`5`的 int 数组，推荐用 `int[] arr = {5}` 或 `int[] arr = new int[]{5}`；
3. 动态初始化的核心是 “先指定长度，后赋值”，写法为 `int[] arr = new int[长度]`，再通过下标赋值。
