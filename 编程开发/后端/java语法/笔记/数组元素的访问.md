## 2— 索引
为了方便对数组元素的操作，给每个元素都赋予了对应的下标
这就叫作索引
简单：索引 数组元素的下标
![[附件存储/f16b903b8a3a63c2f839245536a3c259_MD5.png]]
索引就是数组的一个编号，也叫作：角标，下标，编号
特点 ：从0开始，连续+1,不间断
![[附件存储/16ee4e1af9da3ff59cd19fd95f6232f5_MD5.png]]
只是各自的编号，与里面实际1. 数组的是一个连续的空间
2. 数组一旦定义完毕长度是不可变数据无关
