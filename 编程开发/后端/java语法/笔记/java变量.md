### 2.2— 用法
#### 2.2.1— 格式

1. 重复使用某个值
2. 某个数据经常发生改变
## 3— 注意事项
1. 变量值同时只能有一个
2. 变量名不允许重复定义
3. 一条语句可以定义多个变量
4. 变量在使用之前一定要进行赋值
5. 注意变量作用域范围，只属于当前大括号
