## 1— 内存
软件在运行时，用来临时存储数据的

软件 中输入的数据会临时存储在内存中，只有点击保存
才会存储在硬盘中
![PixPin_2026-03-05_10-03-26.gif](https://obsidian-1332276919.cos.ap-guangzhou.myqcloud.com/20260305100451982.gif)

临时存储数据，那有两种操作是避免不了的
1. 将数据保存在内存中
2. 从内存的对应位置把数据取出来
内存是32g 的如何确定数据的位置呢
## 2— 内存地址
将内存分为一个个的小格子，并给予独立编号
独立编号就是内存地址
作用就是方便我们快速管理内存空间
