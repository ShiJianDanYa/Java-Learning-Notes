# 1— 需要解决问题
知道具体数组元素情况下，如何创建数组
# 2— 本质 意义 解决方法
## 1— 静态初始化
初始化：是指在定义变量，数组，对象的时候进行赋值
初始化：在定义的同时进行赋值
简单：初始化 定义和赋值同时进行
![[附件存储/749d63100348e2bc1bbd1d8d7a36b07f_MD5.png]]
静态：在定义变量，数组，对象的时候，数据是静止的，确定的
简单：静态 定义时数据是确定的

数组的静态初始化：创建数组的时候直接给数组元素赋确定的字面量值
简单：数组初始化 定义数组的同时给其赋确定的值
