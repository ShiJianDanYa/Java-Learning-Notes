## 4— 知识扩展
**Standard Editor（标准版）** J2SE 包含构成 Java 语言核心的类，定位在个人计算机上的应用。
这个版本是Java平台的核心，它提供了非常丰富的API来开发一般个人计算机上的应用程序，包括用户界面接口AWT及Swing，网络功能与国际化、图像处理能力以及输入输出支持等。在上世纪90年代末互联网上大放异彩的Applet也属于这个版本。Applet后来为Flash取代，Flash即将被HTML5取代。
比如：数据库连接、接口定义、输入输出、网络编程等。

**Java ME（Micro Edition）** 是 Java 平台的精简版，主要面向嵌入式设备和早期功能手机开发。虽然在智能手机普及后，它的主流市场被 **Android** 和 **iOS** 取代，但在一些特定领域依然有生命力，例如 **物联网设备、老旧功能机、特定企业系统** 等
