## 1— 继承
重复的部分抽取出来组成一个类,作为父类
被抽取的作为子类 继承父类就好了
![[附件存储/b31030658d89e6126cb68b178247c4a2_MD5.gif]]

### 1.1— 什么是继承
面向对象的三大特征之一,继承是类与类之间的一种父子关系,java中提供关键字extends,用于建立类与类之间的关系
![[附件存储/7f59e8b7760dfaf0b9bd62f6ad900a86_MD5.png]]
继承的作用: 将多个子类中的重复代码抽取到父类,子类可以直接使用

### 1.2— 继承好处
* 可以把多个子类中重复的代码抽取到父类中了,提高代码复用性
* 子类可以在父类的基础上,增加其他的功能,使子类更加强大

![[附件存储/69e61d389e6ab8f5e941f302f93efcce_MD5.png]]
### 1.3— 继承特点
java只支持单继承，不支持多继承，但支持多层继承
单继承：一个子类只能继承一个父类
不能多继承：子类不能同时继承多个父类
为什么不能多继承 如图所示 多个父类若方法相同，子类调用相同方法时，该如何判定使用的方法，是那个父类
![[附件存储/4dda4c0d2034382f9e9c6965b3012099_MD5.png]]


多层继承：可以嵌套 爷爷被爸爸继承 ，而爸爸被儿子继承

细节一
直接父类 中间没有间隔 爷爷和爸爸 爸爸和儿子
间接父类 中间有间隔 儿子和爷爷（存在爸爸作为中介结点）
![[附件存储/08d8bc9f7921fb6d919264fc03689ead_MD5.png]]
