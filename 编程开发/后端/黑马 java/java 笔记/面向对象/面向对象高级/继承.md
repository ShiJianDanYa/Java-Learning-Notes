## 1— 继承

### 1.1— 什么是继承
继承是类与类之间的一种父子关系,java中提供关键字extends,用于建立类与类之间的关系
![image.png](https://obsidian-1332276919.cos.ap-guangzhou.myqcloud.com/20260318154202823.png)
### 1.2— 继承好处
* 可以把多个子类中重复的代码抽取到父类中了,提高代码复用性
* 子类可以在父类的基础上,增加其他的功能,使子类更加强大
