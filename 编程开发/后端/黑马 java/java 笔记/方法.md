### 3.1— 方法的第一种格式
![[附件存储/6962913893dbf2430efb57d0ec64cc9c_MD5.png]]


调用
格式
```java
方法名（）;
```

![[附件存储/41d925f429ce66918d66d3199e965955_MD5.png]]

<font color="#ff0000">注意方法需要先定义在调用，否则会报错</font>

```java
/*  
简单方法的声明与调用  
 */public class method01 {  
    public static void main(String[] args) {  
        // 调用方法 不调用，不执行  
        //方法名（）;  
        method();  
        System.out.println("程序结束");  
    }  
  
    // 定义一个方法 public static void 方法名（）  
    // {  
    // 方法体  
    // }  
    public static void method(){  
        // 方法体  
        System.out.println("方法体");  
    }  
}
```

<font color="#c00000">看到方法进入方法，执行完毕回到调用处</font>

### 3.5— 方法注意事项 ⭐⭐⭐（<font color="#d99694">煮衣服的石像</font>）
1. 方法不调用，就不执行
2. 方法与方法之间是平级关系，不能互相嵌套定义
3. 方法的编写顺序与执行顺序无关
4. 返回值为void为没有返回值 retuen可以省略不写，但如果要编写return，后面就不能跟具体的数据
5. return后面是不能编写代码，因为永远执行不到

### 3.7— 方法的重载 （超重载物）  ⭐⭐⭐⭐

* 同一个类中，定义了多个同名的方法，这些同名的方法具有相同的功能
* 每个方法具有不同的参数类型或参数个数，这些同名的方法，就构成了重载关系

我的理解 <font color="#6425d0">类名相同 ，名字相同，参数类型和个数不同的方法他们之间构成了重载</font><span style="background:#ff4d4f">与返回值无关</span>

参数不同 <font color="#6425d0">个数</font> <font color="#6425d0">类型</font> <font color="#d83931">顺序</font>

顺序不同可以构成重载，但是不建议

java虚拟机通过<font color="#ff0000">参数不同来区分同名的方法</font>

#### 3.7.1— 定义方法的重载 ⭐⭐<font color="#d99694">定蚁</font>

一般是相同功能，取一样的名字  
1.定义方法可以不用太多单词  
2.调用时也不用那么多脑筋


```java
/* 要求使用重载的方法实现如下功能：  
判断两个整数的大小  
兼容所有整数类型（byte,short,int,long）  
 */public class HeavyLoad01 {  
    public static void main(String[] args) {  
        method(10,20);  
        System.out.println("程序结束");  
    }  
    //方法重载  
    /*  
    一般是相同功能，取一样的名字  
    1.定义方法可以不用太多单词  
    2.调用时也不用那么多脑经  
     */    
     public static void method(byte a,byte b){  
        if(a==b){  
            System.out.println("两个数相等");  
        }else{  
            System.out.println("两个数不相等");  
        }    }  
    public static void method(short a,short b){  
        if(a==b){  
            System.out.println("两个数相等");  
        }else{  
            System.out.println("两个数不相等");  
        }    }  
    public static void method(int a,int b){  
        if(a==b){  
            System.out.println("两个数相等");  
        }else{  
            System.out.println("两个数不相等");  
        }    }  
    public static void method(long a,long b){  
        if(a==b){  
            System.out.println("两个数相等");  
        }else{  
            System.out.println("两个数不相等");  
        }    }  
}
```
