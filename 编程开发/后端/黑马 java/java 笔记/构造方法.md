# 构造方法
## 构造方法作用
构造方法也叫构造器，构造函数
作用：<font color="#6425d0">在创建对象的同时
给成员变量进行赋值的</font>

![[附件存储/10606f5b609bde9098fc8eae82db887b_MD5.png]]
## 构造方法的格式
![[附件存储/e81f1e99f17358a193c4fab94a36898e_MD5.png]]

```java
修饰符 类名(参数){
	方法体;
}
```
## 构造方法特点
1. 方法名与类名相同，大小写也要一致
2. 没有返回值类型，连void都没有
3. 没有具体的返回值（不能由retrun带回结果数据）
## 构造方法执行时机
1. 创建对象的时候由虚拟机调用，不能<font color="#6425d0">手动调用构造方法</font>
2. 每创建一次对象，就会调用一次构造方法
## 构造方法的注意事项  ⭐⭐⭐
![[附件存储/141b4931b84b4d875e0bcdfd8dcef4de_MD5.png]]
### 构造方法的定义
* 如果没有定义构造方法，系统将给出一个默认的无参数构造方法
* 如果定义了构造方法，系统将不再提供默认的构造方法

![[附件存储/5b2e55403e655dea13b0d9b92bb931e3_MD5.png]]

![[附件存储/52b515fbcadc6d84ce17c99f509b8bea_MD5.png]]
### 构造方法的重载
* 带参构造方法，和无参数构造方法，两者方法名相同，但是参数不同，这叫做构造方法的重载
### 推荐使用的方式
* <font color="#d83931">无论是否使用，都手动书写无参数构造方法，和带全部参数的构造方法</font>
