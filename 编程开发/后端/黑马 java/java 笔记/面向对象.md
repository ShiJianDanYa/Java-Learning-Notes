## 设计对象并且使用 ⭐⭐
### 类和对象 ⭐⭐
<font color="#6425d0">类（设计图）</font>：是对象共同特征的描述
<font color="#6425d0">对象</font> ：是真实存在的具体东西

<font color="#6425d0">类就是设计图（型号),对象就是根据设计图所制造出来的具体型号（就是你所new出来的东西）</font>

<font color="#d83931">在java中必须先设计类，才能获得对象</font>
#### 类的格式 ⭐
![image.png](https://obsidian-1332276919.cos.ap-guangzhou.myqcloud.com//obsidian20250405223646083.png)

定义类
```java
public class 类名{
	成员变量；
	成员方法；
}
```

对象
```java
类名 对象名 = new 类名（）;
```

使用 对象
```
访问属性 对象名.成员变量
访问行为 对象名.方法名（）
```

##### 定义类 注意事项
![image.png](https://obsidian-1332276919.cos.ap-guangzhou.myqcloud.com//obsidian20250405224448556.png)

* 类名首字母建议大写，需要见名知意，驼峰模式。
* 一个 Java 文件中可以定义多个 class 类，且只能一个类是public修饰，而且<font color="#ff0000">public修饰</font>的<font color="#ff0000">类名</font>必须成为<font color="#ff0000">代码文件名</font>。实际开发中建议还是一个文件定义一个class类。
* 成员变量的完整定义格式是：修饰符 数据类型 变量名称 = 初始化值；一般无需指定初始化值，存在默认值。

### 键盘录入两套体系 ⭐⭐

#### 第一套体系： ⭐

nextInt（）；接收整数
nextDouble();接收小数
next（）;接收字符串

遇到空格，制表符，回车就停止接受(此时回车还在内存中)。这些符号后面的数据就不会接受了

#### 第二套体系： ⭐⭐

nextLine（）;接收字符串

可以接收空格，制表符，遇到回车才停止接受数据


#### 两套不能混用
