# 标注的javaBean

㊀ 类名需要见名知意
㊁ 成员变量使用<font color="#ff0000">private</font>修饰
㊂ 提供至少两个构造方法
* 无参构造方法
* 带全部参数的构造方法成员方法
㊃ 成员方法
* 提供每一个成员变量对应的 <font color="#ff0000">setXxx（）/getXxx</font>
* 如果还有其他行为，也需要写上
