### 2.2— 自增自⚔️运算符

| 符号  | 作用  | 说明    |
| --- | --- | ----- |
| ++  | 加   | 变量值+1 |
| --  | ⚔️  | 变量值-1 |

#### 2.2.1— 注意事项
++ -- 可以在变量前面也可以在后面

1. 两者在单独一行没有区别
2. ++--在前 先自增自减，再参与运算
3. ++--在后 先参与运算，再自增自减
