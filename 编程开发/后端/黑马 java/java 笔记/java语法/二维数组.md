## 1— 二维数组
### 1.1— 二维定义
数组中的每一个元素都是一个一维数组,这个数组就是一维数组.(就是数组里面包数组)
### 1.2— 初始化
#### 1.2.1— 静态初始化
格式
```java
// 数据类型[][] 数组名 = new 数据类型[][]{元素1,元素2,元素3,......}
int[][] arr = new[][]{{12,24,36},{666,888},{10,20,30},{999}};
```

### 1.3— 动态初始化
格式
```java
// 数据类型[][] 数组名 = new 数据类型[长度1][长度2]
int[][] arr = new int[3][5];
```
动态化数组也是有数据的
