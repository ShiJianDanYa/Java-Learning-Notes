如果满足对应条件，则执行对应的代码,否则不执行
# 2— 本质 意义 解决方法
## 1— 单分支判断语句
### 1.1— 判断语句格式
```java
if(关系表达式){
	语句体;
}
```

# 3— 注意事项
if判断语句 细节
1. If语句大括号的省略 如果大括号中语句体只有一行，大括号可以省略
2.  小括号后面不能有分号，这样会拆开if的语句结构
3. If语句大括号的位置 左括号写在上一行的末尾，不要单独写一行
4. 判断布尔类型的变量 判断布尔类型的变量，直接把变量写在小括号中即可

## 2— 小括号后面不能有分号

```java
if(true);{
System.out.println("你好啊世界");
}
```
最终无论 （）内的是否成立
你好啊世界 语句都会输出 

因为Java把代码理解为
```java
//分号;判断---语句体之间的联系切断
if(true);
//独立的代码快，不受上面的if控制
System.out.println("语音警告，当前体温已经超过了38度---1");

```

## 3— 布尔类型的变量
布尔类型的变量 判断布尔类型的变量，为什么直接把变量写在小括号中
![[附件存储/58f55d361463c8496e78ff22ad2bca1c_MD5.png]]
因为要避免 b=true 情况出现
这样无论 b 原值无论如何 语句都会执行
