# 1— 需要解决问题
根据多种条件，执行满足对应条件的语句
# 2— 本质 意义 解决方法
多条件判断语句 有两种

格式
```java
if(关系表达式A){
	语句体A;
} else if(关系表达式B){
	语句体B;
} else if（关系表达式C）{
	语句体C;
}
...
else {
语句体 N;
}
```
流程
从上往下依次进行判断
判断成立，执行语句体
都不成立
执行else
