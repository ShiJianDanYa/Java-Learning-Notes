### 1.1— 大小关系 ⭐️⭐️⭐️


```java
  
public class data_type {  
    public static void main(String[] args){  
        //byte  
        byte b = 10;  
        System.out.println(b);  
        //short  
        short s = 20;  
        System.out.println(s);  
        //int  
        int i = 30;  
        System.out.println(i);  
        //long  
        //定义long类型的变量  
        //需要在后面加l作为后缀  
        //这里和下面的float
        //大小写没有限制，但为了便于区分建议打大写  
        long n = 99999999999L;  
        System.out.println(n);  
  
        //float  
        //后缀 f        
        float f = 10.1f;  
        System.out.println(f);  
        //double  
        double d = 20.5;  
        System.out.println(d);  
  
        //char  
        char c = '中';  
        System.out.println(c);  
  
        //boolean  
        //true false 
        boolean o = true;  
        System.out.println(o);  
  
  
    }
```

### 1.2— 基本数据类型注意事项
无大小写限制

1. 定义long类型的变量，需要在后面加L作为后缀
2. 定义float类型的变量，需要在后面加f作为后缀
