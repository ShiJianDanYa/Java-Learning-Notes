# 2— 本质 意义 解决方法
多分支判断语句
格式
```java
if(关系表达式A){
	语句体A;
} else if(关系表达式B){
	语句体B;
} else if（关系表达式C）{
	语句体C;
}
...
else {
语句体 N;
}
```
