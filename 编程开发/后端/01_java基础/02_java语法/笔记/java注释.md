## 1— 需要解决问题
经历一段时间后，遗忘代码功能和编程思路
## 2— 本质 意义 解决方法
### 2.1— 注释
给代码添加的解释说明的性的信息
### 2.2— 注释种类 ⭐️⭐️
1. <font color="#ff0000">单行注释 //</font>
2. <font color="#ff0000">多行注释 /* */</font>
3. 文档注释/**  \*/ 
## 3— 注意事项
1. 注释的内容不会参与编译，运行
2. 多行注释，不支持嵌套
## 4— 知识扩展
注释快捷键
单行注释快捷键 ctrl + / 
多行住注释快捷键 ctrl + shift + /
多行注释为什么不支持嵌套
![image.png](https://obsidian-1332276919.cos.ap-guangzhou.myqcloud.com/20260307185438004.png)
因为注释的逻辑是/* 开始是注释 寻找到*/ 才是一个注释
结束
但是 如果 这个注释里有 /* 只会把他当作注释内容
不会当作注释开头，那另一边的*/ 就是一个没有头的多行注释
自然会报错
