# 1— 需要解决问题
if语句在解决单变量匹配多个固定值时，语法繁杂，较难理解
# 2— 本质 意义 解决方法
选择语句：把所有的选择一一列举出来，根据不同的条件<font color="#d83931">任选其一</font>
格式
```java
switch(week){
	case 值1:
		语句体1;
		break;
	case 值2:
		语句体2;
		break;
	...
	default:
		语句体n;
		break;
}
```
这里的<font color="#d83931">week</font>是一个表达式(是一个<font color="#d83931">具体数值</font>),不是一个判断

执行流程
1. 首先计算表达式的值。
2. 依次和case后面的值进行比较，如果有对应的值，就会执行相应的语句，在执行的过程中，遇到break就会结束。
3. 如果所有的case后面的值和表达式的值都不匹配，就会执行default里面的语句体，然后结束整个switch语句。

# 3— 注意事项

1. 表达式：结果（字符/整数byte short int/枚举(jdk5)/字符串(jdk7)）
2. case：被匹配的值，只能是真实的数据
3. case：值不允许重复
4. break：表示中断，结束的意思，结束switch语句
5. default：所有情况都不匹配，执行该处的内容

# 1— 知识扩展下 
## 1— 注意事项一为什么不能是其他值

跳转表，索引不支持小数，
也不支持大的整数long

## 2— default的位置和省略
位置 case和default是没有标准的的上下之分的，位置可以任意的书写
为了方便观看，提高代码的阅读性
一般来讲，case是从下到大书写的，default是写在最下面的。
default也可以省略不写，在此时如果所有的case都不匹配，则没有任何输出结果
## 3— case穿透
在我们书写代码的时候，如果break没有写，此时就会触发case穿透问题。
执行流程
1. 拿着小括号中的表达式的值跟下面的case进行匹配
2. 如果匹配符合，就会执行case里面的语句体，遇到break结束整个switch（正常情况）
3. 如果执行语句体的时候没有遇到break，那么程序会继续执行下一个case的语句体，直到遇到break或者运行完整的switch为止

应用场景：
当多个case的语句体重复的时候，利用case穿透可以节省代码。

### 4.1— 箭头标签
源代码
```java
switch(number)
{
	case 1:
		System.out.println("一");
	break;
	
	case 2:
		System.out.println("二");
	break;
	
	case 3:
		System.out.println("三");
	break;
	
	default:
		System.out.println("没有这个星期");
	break;
}
	
```
通过箭头简化后
```java
switch(number)
{
	case 1 ->{
		System.out.println("一");
	}

	case 2->{
		System.out.println("二");
	}
	
	case 3->{
		System.out.println("三");
	}
	
	default->{
		System.out.println("没有这个星期");
	}
	
}
```
简化后 自带 break;
只有一个语句{ } 也可以省略
最终简化版本
```java
switch(number){
case 1 ->System.out.println("一");
	case 2->System.out.println("二");
	case 3->System.out.println("三");
	default->System.out.println("没有这个星期");
}
```
### 4.2— case后面可以写多个值
条件不同，但是执行语句相同，那么就可以在case后面写多个值,用逗号隔开。
```java
switch(number){  
    case 1,2,3,4,5 ->System.out.println("工作日");  
        default->System.out.println("没有这个星期");  
}
```
### 4.3— switch可以有运行结果
我们如果后面需要用到switch运行结果呢？
```java
//需要一个变量接受数据  
String name =switch(number){  
    case 1,2,3,4,5->{  
        yield"工作日";  
    }  
    default->{  
        yield"没有这个星期";  
    }  
//大括号后面还需要补一个分号  
};
```

还是太复杂了能不能在简化一下
```java
//需要一个变量接受数据  
String name =switch(number){  
    case 1,2,3,4,5 ->"工作日";  
        default->"没有这个星期";  
//大括号后面还需要补一个分号  
};
```
### 4.4— yield关键字
`yield` 用于在 `switch` 表达式中显式返回值
