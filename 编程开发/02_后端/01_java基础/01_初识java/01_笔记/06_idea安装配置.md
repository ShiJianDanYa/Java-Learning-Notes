## 2— idea安装
第一步
双击idea安装包
![[idea安装配置_2026-03-30_19-32-24.png]]
第二步 开始安装
安装包加载完毕，进入安装程序，点击右下角的下一步
![[idea安装配置_2026-03-30_19-35-06.png]]
第三步 选择安装位置
选择安装位置，默认是C盘，如果想修改可以点击后面的浏览
![[idea安装配置_2026-03-30_19-41-48.png]]
规则
* 定义一个文件夹专门用来安装跟开发相关的所有软件
* 整个路径上不要有中文，不要有空格，不要有特殊符号
	* 正确示例：F:\develop\idea025\Intellij
	* 错误示例：F:\开发\idea025\Intellij]
		* 错误点：整个路径上不能有中文，担心记不住英文单词的兄弟可以用拼音
第四步 相关设置
此时只要在创建桌面快捷方式下面打勾即可，其他地方不需要勾选，点击下一步
![[idea安装配置_2026-03-30_19-51-28.png]]
第五步 等待安装
展示开始的内容，此界面不需要做任何操作，直接点击右下角的安装即可
系统会开始安装，等待1~5分钟即可(时间跟电脑配置有关）
![[idea安装配置_2026-03-30_19-55-27.png]]
![[idea安装配置_2026-03-30_19-57-48.png]]
第六步 安装完毕
安装完毕点击完成
中间的运行，勾不勾都可以
* 勾选上，点击完成之后会自动打开idea
* 不勾上，点击完成之后不会自动打开idea
![[idea安装配置_2026-03-30_20-07-06.png]]
第七步 检查安装是否成功
回到桌面，看是否有idea的快捷图标，如果有，就代表安装成功
## 3— idea删除历史记录
IDEA是如果电脑上已经安装过老版本的IDEA，可以卸载，也可以用老版的。如需卸载，一定要卸载干净。
如果电脑是第一次安装，直接跳过
删除记录图解
第一步卸载老版本IDEA
注意：卸载的时候勾选上历史记录和安装插件
![[idea安装配置_2026-03-30_20-13-07.png]]
第二步 清除电脑上两个位置的记录
第一个位置在：C：\Users\自己电脑名\AppData\Local\JetBrains，把JetBrains文件夹全部删除
第二个位置在:8C:\ProgramData\Microsoft\Windows\Start Menu\Programs\JetBrains把JetBrains文件夹全部删除
注意 AppData是隐藏文件夹，一定要上面操作先显示电脑隐藏文件夹，否则无法看到该文件夹
