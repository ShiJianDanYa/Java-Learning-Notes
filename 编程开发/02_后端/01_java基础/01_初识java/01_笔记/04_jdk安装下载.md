# 2— 核心内容
### 0.1— jdk
#### 0.1.1— JDK下载
官方网址下载:[jdk官网地址](https://www.oracle.com/cn/)
注意1：针对不同的操作系统，需要下载对应版本的JDK。
注意2：如果你的电脑是windows32位的，建议重装系统，重装成64位的操作系统。因为Java从9版本开始，就已经不提供32位版本的安装包了。
#### 0.1.2— JDK安装
傻瓜式安装，下一步即可。默认的安装路径是在C:\Program Files\Java\jdk-24\
#### 0.1.3— 安装目录

| 目录名称    | 说明          |
| ------- | ----------- |
| bin     | 存放各种工具命令    |
| conf    | 存放相关配置文件    |
| include | 存放平台特定头文件   |
| jmods   | 存放各种模块      |
| legal   | 各种模块的授权文档   |
| lib     | 工具的一些补充jar包 |
### 0.2— 环境变量
#### 0.2.1— 配置环境变量作用
如果我想要在CMD的任意目录下，都可以启动某一个软件，那么就可以把这个软件的路径配置到环境变量中的PATH里面。
在启动软件的时候，操作系统会先在当前路径下找，如果在当前路径没有找到，再到环境变量的路径中去找。如果都找不到就提示无法启动。
简单:在任意目录下都能打开指定的软件
