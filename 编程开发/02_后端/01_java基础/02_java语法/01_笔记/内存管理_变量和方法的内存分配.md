## 1— 虚拟机分区
虚拟机将其运行时占用的内存分为五块区域
1. 栈内存 方法被调用进栈执行，执行完毕出栈 和<font color="#d83931">方法有关</font>
2. 堆内存 所有线程共享，存储对象，数组，和字符串常量池  只需要记住和<font color="#d83931">new有关</font>
3. 方法区 JDK7 [[内存管理_变量和方法的内存分配#^136dbf|永久代]] 实现，JDK8开始 元空间 实现，从虚拟机内部移到本地内存。存储字节码信息、常量、静态变量  <font color="#d83931">字节码信息</font>
4. 本地方法栈 调用本地Native方法
5. 程序计数器 每个线程独立、记录当前线程执行的字节码指令地址（行号）
